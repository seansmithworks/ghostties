Support

Ghostties is in beta. If something breaks or feels wrong, we'd like to know.

Found a bug?

Open an issue on GitHub. Include your macOS version (Apple menu → About This Mac), Ghostties version (Ghostties menu → About Ghostties), and what you were doing when it broke.

System requirements

  • macOS 13 (Ventura) or later
  • Apple Silicon Mac (M1, M2, M3, M4) — universal binary not available yet

Updates

Ghostties auto-updates via Sparkle. You'll get a prompt when a new release is ready. To disable updates, add auto-update = off to your config.

Reset to factory state

Quit the app, delete ~/Library/Application Support/com.seansmithdesign.ghostties/, then relaunch. This wipes workspaces and preferences but leaves your project-local task notes (.ghostties/tasks/) alone.

Contact

For now, GitHub Issues is the best path. Email and Discord support coming once we're out of beta.
